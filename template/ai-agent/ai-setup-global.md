# AI設定生成器（グローバル／端末用）

このファイルをAIエージェントに読み込ませると、端末レベルの個人共通設定を生成できる。
**ツール非依存・プロジェクト非依存の個人ルールだけ**を対象とする（ここを肥大化させないのが鉄則）。

> 使い方：dotfiles等に置き、端末セットアップ時にエージェントへ
> 「`ai-setup-global.md` を読んでグローバル設定を生成して」と指示する。
> 複数端末で使う場合、このファイルをdotfilesで同期すれば各端末で同じ設定を再生成できる。

---

## 0. グローバルに書いてよいもの／いけないもの

- 書いてよい：応答言語・口調・単位・通貨・コミット規約・破壊的操作の方針など、
  どのプロジェクトでもどのツールでも変わらない個人設定。
- 書いてはいけない：プロジェクト固有のスタックや受け渡し規約（→ ローカル側へ）、
  MCP設定・permissions（→ ツールごとのconfigへ。概念がツール間で異なり共有不可）。

---

## 1. 生成するファイルと構成

| ツール      | グローバル指示書のパス                                                                 |
| ----------- | -------------------------------------------------------------------------------------- |
| Codex       | `~/.codex/AGENTS.md`（**マスター＝実体**）                                             |
| Claude Code | `~/.claude/CLAUDE.md`（中身は `@~/.codex/AGENTS.md` の1行のみ）                        |
| Gemini CLI  | `~/.gemini/settings.json` に `"contextFileName": "AGENTS.md"` を設定（別ファイル不要） |

- マスターは `~/.codex/AGENTS.md` に一本化し、他ツールは import で参照する。
- symlink方式（`ln -s` 等）はMac中心なら可だが、Windowsは管理者権限／開発者モードが要るため
  import方式（`@path` 1行）を既定とする。OS非依存で複数端末に強い。

---

## 2. マスター（~/.codex/AGENTS.md）に含める内容

以下をそのまま生成する。これらはツール非依存の個人ルール。

```markdown
# Global Rules

## 応答

- 応答は日本語。
- 時刻はJST（UTC+9）・24時間制（24時超えは25時・26時表記も可）。
- 単位はメートル法。通貨は日本円（¥）基本、海外製品・サービスは元通貨も併記。

## 口調・スタイル

- 基本はフランクで親しみやすく、技術・専門的な話題ではビジネスライクに切り替える。
- 明瞭・簡潔・実用性を最優先。冗長な説明・歯切れの悪い表現・根拠のない称賛は避ける。
- 率直でストレートに。同意一辺倒ではなく、異なる視点や懸念点も積極的に提示する。

## 開発の作法

- コミットメッセージは Conventional Commits（feat: / fix: / refactor: 等）。
- 破壊的操作（rm, force push, 本番DB更新など）は必ず事前確認する。
- 不明点は推測で進めず質問する。
```

---

## 3. ラッパー（~/.claude/CLAUDE.md）の中身（固定）

```markdown
@~/.codex/AGENTS.md
```

---

## 4. 生成手順（エージェントへの指示）

1. `~/.codex/` `~/.claude/` が無ければ作成する。
2. `~/.codex/AGENTS.md` を §2 の内容で生成する（既存があれば差分提案）。
3. `~/.claude/CLAUDE.md` を §3 の内容で生成する。
4. Gemini を使う場合のみ `~/.gemini/settings.json` に `contextFileName` を追記案内する。
5. プロジェクト固有の事項は一切書かない。必要ならローカル用生成器（`ai-setup-local.md`）へ誘導する。

---

## 5. グローバルとローカルの責務分担（早見表）

| 項目                           | グローバル | ローカル（リポジトリ） |
| ------------------------------ | ---------- | ---------------------- |
| 応答言語・口調・単位・通貨     | ○          | －                     |
| コミット規約・破壊的操作の方針 | ○          | （上書きしたい時のみ） |
| プロジェクトのStack            | －         | ○                      |
| 受け渡し規約・委譲の出力契約   | －         | ○                      |
| MCP設定・permissions           | ×          | × （各ツールconfig）   |
