# note スキル

後続作業で再利用する調査結果や判断記録を `.agents/note/` に保存する。

## 手順

1. 記録すべき内容を整理し、`.agents/note/` にファイルを作成する。
2. ファイル名はケバブケースの短い名前（例: `foo-bar.md`）。

## ファイル構成

```markdown
# タイトル

## 結論

判断・結論を先に。1〜3文。

## 経緯

調査内容や比較検討の詳細。
```

## 注意

- 一時的な情報やセッション内で完結する内容は note にしない。
- 関連する plan があればファイル内でリンクする。
